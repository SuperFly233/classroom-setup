连接 GitHub
需要 Personal Access Token 来读写仓库。
前往 GitHub → Settings → Tokens (classic)， 勾选 repo 后生成，粘贴到下方。
Token 仅存储在本机浏览器 localStorage，不上传任何服务器。
格式：ghp_xxxxxxxxxxxxxxxx
